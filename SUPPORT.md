# Support

Use the right channel so maintainers can respond quickly.

| Need | Where To Go |
|---|---|
| Usage question | Open a Question issue |
| Bug report | Open a Bug Report issue |
| Feature idea | Open a Feature Request issue |
| Protocol change | Open an RFC Proposal issue |
| Security vulnerability | Email security@aifinpay.io |
| Documentation issue | Open a pull request or documentation issue |

## Before Asking

Please check:

- `README.md`
- `docs/index.md`
- `docs/aifp/README.md`
- `docs/aifp/01-AIFP-1-RFC-Payment-Protocol-Specification.md`
- `docs/aifp/07-Quick-Start-Guide.md`
- `docs/aifp/08-OpenAPI-3.1-Specification.yaml`

## Useful Context

For technical questions, include:

- Merchant or agent role.
- SDK language and version.
- Protocol version.
- Request id if available.
- Relevant endpoint.
- Expected and actual behavior.
