# Quick Start

Choose the path that matches your runtime.

## Paths

| Path | Best for |
|---|---|
| [Node / TypeScript](node-typescript.md) | Browser, server, or worker apps in JavaScript/TypeScript |
| [Python](python.md) | Backend services, notebooks, and automation |
| [MCP Server](mcp-server.md) | Connecting developer tools through an MCP server |

## Default Mode

Sandbox is the default. Use sandbox/testnet keys, sandbox URLs, and faucet-backed tokens first.
Only move to production after you have read [x402 Flow](../core-concepts/x402-flow.md),
[Security Model](../security-model.md), and [Error Codes](../reference/error-codes.md).

## Package Status

The quickstarts reference alpha package names for the planned SDK and MCP surfaces. Availability
and version badges are published with each package release.
