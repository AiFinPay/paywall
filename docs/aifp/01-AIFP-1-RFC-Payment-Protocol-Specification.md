# AIFP-1 — AiFinPay Payment Protocol Specification

> Canonical source path reserved for the full AIFP-1 RFC.

This document is the normative specification for AiFinPay Paywall Protocol (AIFP). It governs all other documents and defines the HTTP `402 Payment Required` flow, payment challenge, receipt token, merchant/agent/wallet behavior, error model, settlement, security requirements, and conformance rules.
